# My Portfolio

A personal portfolio website built with React, Vite, and Material UI.

It includes:

- Profile and contact links
- Skills
- Work experience
- Projects
- Education
- Resume download

## Local Development

Install dependencies:

```bash
npm install
```

Start the local dev server:

```bash
npm run dev
```

Build the production version:

```bash
npm run build
```

## Deploy

To deploy new changes, switch to the main branch first, then run:

```bash
npm run predeploy
npm run deploy
```
